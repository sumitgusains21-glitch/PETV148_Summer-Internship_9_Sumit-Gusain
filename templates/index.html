Security Features

NIST SP 800-63B

Evaluates password length and checks against common password lists following modern NIST recommendations.

OWASP Guidelines

Verifies password complexity using OWASP Password Storage and Authentication best practices.

Common Password Detection

Compares the entered password against a database of commonly used passwords to detect weak credentials.

bcrypt Security

Passwords are never stored in plain text. Secure bcrypt hashing is used before logging audit information.

Security Score

Calculates an overall compliance score out of 100 based on multiple password quality parameters.

Audit Logging

Every audit is securely recorded inside the SQLite audit_logs table for future analysis.

About This Project

This Password Policy Compliance Auditor is a Cybersecurity Project developed using Python, Flask, SQLite and bcrypt. It evaluates passwords against industry-recognized standards including NIST SP 800-63B and OWASP Password Guidelines, providing users with a detailed security assessment and actionable recommendations.
